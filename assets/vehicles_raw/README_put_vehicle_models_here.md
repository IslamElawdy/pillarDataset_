# Raw vehicle models

Put legally usable vehicle models here.

Supported target formats for future import workflow:

- `.blend`
- `.glb` / `.gltf`
- `.fbx`
- `.obj`

Important:

- Do not commit large models to Git unless intentionally using Git LFS.
- Check commercial/research license terms for every model.
- Prefer generic vehicle models without manufacturer logos.
- Remove number plates, brand badges, and trademarked details if needed.

The project can run without models using the procedural fallback generator.
